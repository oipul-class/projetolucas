Sistema Operacional Código Aberto

Definição:

O código aberto (Open Source) é um sistema que permite que o código fonte de software possa ser utilizado, estudado e modificado por qualquer usuário sem alguma autorização prévia, mas o desenvolvedor original do software determina as condições de uso e distribuição. É comum achar que o código aberto é o mesmo que software livre, mas não é. Qualquer licença de software livre é também uma licença de código aberto, mas o contrário não é necessariamente verdade, isso acontece porque o código aberto não compartilha das quatro liberdades essenciais definidas pela Free Software Foundation, que são:

  • Executar o software como desejar, para qualquer propósito;
  • Estudar o software e adaptá-lo como quiser;
  • Redistribuir cópias para ajudar outros;
  • Distribuir cópias de suas versões modificadas.

Modo texto (linha de comando):

Uma interface de linha de comandos, em inglês command-line interface, é um meio de interagir com um programa de computador, onde o utilizador emite comandos para o programa sob a forma de sucessivas linhas de texto.

Navegação entre diretórios

Para navegar em diretórios, precisa-se saber quais têm, então digite o comando “DIR”, que irá exibir todos os seus arquivos. Agora para navegar realmente, o comando “CD” permitirá que o usuário entre em pastas e o “CD..” retornar à pasta anterior.

Criação de diretórios e arquivos:

O comando "MD" cria pastas, dando-te também a possibilidade de criar arquivos, subdiretórios e diretórios. Assim como o comando “MKDIR”, porém, ele apenas cria um diretório.

Renomeação de diretórios e arquivos:

O comando “REN” possibilita que renomeie o nome de diretórios e dos arquivos, assim como o comando “RENAME”.

Exclusão de diretórios e arquivos:

O comando "DEL" deleta as pastas, diretórios e arquivos, assim como o comando “ERASE” que também exclui um ou mais arquivos. Já o comando “RD” propriamente remove um diretório.

Movimentação de diretórios e arquivos:

O comando “MOVE” é o responsável por isso. Ele movimenta arquivos e te dá também a possibilidade de os renomear.

Cópia de diretórios e arquivos

Comando “COPY” copia arquivos e também possibilita copiar mais de um arquivo para outro local.

Modo gráfico (interface gráfica)

Área de trabalho

Após iniciar o Windows, a primeira coisa com que nós nos deparamos é a área de trabalho, também conhecida como desktop, a área na tela em que você trabalha. É nela onde você poderá guardar todas as suas pastas de trabalho contendo seus documentos (desenhos, ofícios, relatórios, planilhas, bancos de dados gráficos, músicas, etc).

Trabalho com janelas

Sempre que um programa é aberto, uma pasta ou um arquivo, aparecerá na tela, em uma caixa ou moldura, uma janela (daí o nome atribuído ao sistema operacional Windows ). Há como movê-las, alterar o tamanho ou simplesmente fazê-las “desaparecer”. Todas as janelas têm algumas coisas em comum, independentemente do que há nelas. Em primeiro lugar, elas sempre aparecem na área de trabalho, a principal área da tela. E suas partes principais são:

  • Barra de título: exibe o nome do documento e do programa.
  • Botões Minimizar, maximizar e fechar: estes botões permitem ocultar a janela, alargá-la para preencher a tela inteira e fechá-la.
  • Barra de menus: contém itens nos quais você pode clicar para fazer escolhas em um programa.
  • Barra de rolagem: permite rolar o conteúdo da janela para ver informações que estão fora de visão no momento.
  • Bordas e cantos: é possível arrastá-los com o ponteiro do mouse para alterar o tamanho da janela.

Configuração do sistema

Essa é uma propriedade de sistemas operacionais de interface gráfica que permite configurar todas as ações necessárias para manter o sistema operacional em funcionamento. Em dispositivos Windows, as configurações do sistema são conhecidas como “msconfig”. Essa ferramenta pode gerenciar aspectos como a carga inicial do sistema e arquivos de inicialização (system.ini, win.ini).

Ferramentas do sistema:

Há várias ferramentas importantes em um sistema operacional que fornecem ao usuário informações e dados técnicos, servindo para a resolução de ocasionais problemas de hardware, falta de memória e problemas de segurança, por exemplo. Algumas dessas ferramentas, tomando Windows como exemplo, são: Desfragmentador de unidades, visualizador de eventos, informações do sistema, diagnóstico de memória do Windows, limpeza de disco, monitor de desempenho, firewall etc. Essas ferramentas podem ser acessadas no menu de pesquisa.





Acessórios:

Os principais acessórios para a manipulação da interface gráfica são o mouse e o teclado. O mouse é um dispositivo indicador que serve para deslocar um cursor na tela, permitindo selecionar, mover e manipular objetos graças a botões. Chama-se clique, a ação de pressionar (clicar) um botão para efetuar uma ação.
O teclado de computador é um dispositivo que possui uma série de botões ou teclas, e é utilizado para inserir dados no computador. É um tipo de periférico de entrada utilizado pelo usuário para a entrada manual no sistema de dados e comandos. Possui teclas representando letras, números, símbolos e outras funções, baseado no modelo de teclado das antigas máquinas de escrever.

Menu de ajuda:

"Alguns sistemas operacionais possuem um menu de ajuda ou um menu de suporte, um dos exemplos é o Windows. Para acessar o menu de suporte do Windows, abra a ferramenta executar e digite " services.msc " e localize o serviço “Ajuda e Suporte”. Dentro desta aba terá opções de ajuda."

Gerenciamento de arquivos:

A parte do sistema operacional que gerencia os arquivos e as operações de arquivos é a gerência do sistema de arquivos. A gerência do sistema de arquivos também cuida da segurança no acesso aos arquivos, garantindo que um usuário não tenha acesso não autorizado a um arquivo que pertence a outro usuário. A gerência de arquivos também cuida do compartilhamento dos arquivos, através de uma rede, por vários processos e usuários do sistema operacional.

Gerenciamento de usuários e permissões:

Grande parte dos sistemas operacionais sofisticados permitem que a máquina tenha reservado espaços de trabalho para diferentes usuários que possam acessar o computador, e fazem isso nos primeiros protocolos de instalação do sistema operacional em si. O Windows, por exemplo, tem esse sistema desde as primeiras versões como o Windows XP. A possibilidade de gerenciar usuários, e as permissões que cada um recebe, permite que o software seja muito mais organizado, pois previne que documentos sejam misturados ou haja excesso de arquivos em uma pasta só, dificultando a acessibilidade de qualquer usuário que entrar. No Windows, o gerenciamento de usuários e permissões é feito pelo painel de controle, na seção “Usuários”. Lá é possível criar usuários, mudar os nomes, e o usuário administrador (criado desde o início da instalação do S.O) pode dar e tirar permissões administrativas de cada usuário.

Níveis de inicialização e finalização:

Após ligar a máquina, é feita várias verificações pela BIOS, em especial uma revisão geral do hardware da máquina. Após isto, é carregado um pequeno programa que está localizado no 1º setor do disco rígido, chamado de Registro Mestre de Inicialização ou MBR. Ele pode ter uma destas duas funções:

  • Achar a partição ativa e carregar o 1º setor desta partição, que tem um pequeno programa com informações sobre o SO que está nesta partição. Ele carregará os componentes básicos, ou carregará um gerenciador de inicialização que pode finalizar o trabalho.
  • O MBR carrega informações diretamente de uma das partições, e pode utilizá-las para carregar SO de qualquer partição.


A segunda opção é mais abrangente, e, portanto, a mais utilizada. Assim, a BIOS utiliza o MBR para carregar um gerenciador de inicialização, podendo inicializar o Linux ou outros sistemas operacionais de qualquer partição ou dispositivo.

Instalação na máquina virtual:

Vários S.Os de código aberto estão disponíveis em softwares de máquinas virtuais, principalmente o virtual box. Mac OS X, Linux e Windows estão entre os principais baixados virtualmente. O procedimento de instalação não é tão difícil, envolvendo passos práticos de entender, basta apenas ter os arquivos necessários para a instalação e conhecer o hardware da sua máquina, para saber se ela suporta todo o desempenho de um disco óptico virtual da mais atual versão desses sistemas operacionais. Para saber sobre como instalar o Mac OS X no VirtualBox, segue os links e as imagens.

Atualização:

A escolha da versão específica de uma máquina Linux é exclusivamente sua, você pode optar por versões mais antigas ou mais modernas, de acordo com o que for mais cômodo. Mas é muito importante ter em mente que as atualizações incluem uma maior variedade de programas, novos funcionamentos e é claro, um maior processamento. Isto tudo acarreta um maior tamanho nas máquinas virtuais, portanto se o seu PC não aguenta rodar um Ubuntu de última versão, instalá-lo só vai gerar complicações. Também é necessário tomar cuidado com versões antigas, pois elas podem ser ultrapassadas e obsoletas, portanto, seu uso acaba sendo inepto e isso pode atrapalhar no desempenho do usuário, dependendo de qual for seu propósito.

Programas:

"Programas consistem em softwares feitos por programadores, com funções específicas para auxiliar no uso e aproveitamento de um sistema operacional. Seja um explorador de arquivos, gerenciador de tarefas, navegador web, galeria de fotos ou reprodutor de vídeos, todos são exemplos de programas voltadas a certas funções que fornecem ao usuário uma maior gama de possibilidades ao usar a máquina, pois tornam seu uso mais abrangente e prático. Certos programas possuem funções fundamentais que não devem faltar ao usuário, tal como explorador de arquivos, terminal e gerenciador de tarefas. Outros programas são um adicional, não são realmente necessários, porém estão ali para auxiliar o usuário sempre que ele precisar, como galeria e bloco de notas. E há os programas baixados pelo próprio usuário para cumprir funções específicas voltadas as necessidades particulares dele, tal como uma IDE, um simulador ou um jogo."
