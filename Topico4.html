Editor de texto

O que é?

Editor de texto é um software como Microsoft Word e WordPad são usados para escrita e formatação de texto que também podem ser usados para programar em certas linguagens como java, html, css e entre outros

Algumas formatações disponiveis em editores de texto

  1. O editor de texto deixa você colocar o tipo e a distancia de divisão entre os parágrafos
  2. A maioria dos editores de texto permitem que o usúario(a)crie tabelas de dados
  3. Os editores também permite que você separe o conteudo em varias colunas
  4. Fonte ou Fonte tipográfica é um padrão, variedade ou coleção de caracteres que podem ser escolhidas para estilizar o texto
  5. Permite que possa ser escrito letras maiúsculas e minúsculas

Algumas inserções disponiveis em editores de texto

  1. Adicionar uma qubra de linha no meio do texto em qualquer ponto
  2. Adicionar e editor anotações entre os textos
  3. Colocar numeração de paginas
  4. Colocar arquvios txt, word e etc no texto
  5. Criar uma planilha nas paginas
  6. Adicionar e personalizar tabelas da maneira como desejar
  7. Permite adicionar imagens de qualquer formato ou tamanho
  8. Marcar um ou vários elementos com uma cor específica
  9. Personalizar tanto o rodapé quanto o cabeçalho
  10. Criar uma página com a função de orientar o leitor
  11. Um índice para figuras orientando o leitor
  12. Verificar se há erros na ortografia
  13. Tipos de arquivos para a impressão
